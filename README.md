# Etch-a-Sketch
A browser-based drawing game built with **HTML, CSS, and JavaScript** for The Odin Project Foundations course.

## Development Status
- [ ] 16×16 grid generated dynamically with JavaScript
- [ ] Hover over squares to “draw” by changing their color
- [ ] Button to choose a new grid size (up to 100×100)
- [ ] Random color mode for each square interaction
- [ ] Progressive darkening effect (each pass increases opacity by 10%)
